== Changelog ==

= 1.3.0 =
* Added category selection in the Gutenberg block and widget.
* Fixed an issue causing the block preview error when category data was missing.
* Improved category data handling with localized scripts.

= 1.2.0 =
* Added the ability to select categories in widgets and blocks.
* Fixed a bug in handling post thumbnails.
* Improved query performance.

= 1.1.0 =
* Added support for multiple custom post types.
* Fixed an issue with the post title display.

= 1.0.0 =
* Initial version of the plugin.
